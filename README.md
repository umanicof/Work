# Work
就職活動のための提出ソースコードです。一時的に公開しています。  
ポートフォリオ：https://game-creators.camp/@umanicof

* NkfLib  
  個人開発で使用している自前のライブラリです。
  各プロジェクトの汎用的な処理を書き溜めていっているので、やや古いソースコードも多く、また機能も断片的です。
* GhostHouse  
  最近のゲームジャムで作成した「ゴーストたちの晩餐会」というゲームのソースコードです。
  アセットなどはライセンス的に提供が難しいのでソースコードのみの提供となります。
  特徴的な部分としては全体的なゲームスピードを制御する仕組みがあったり、負荷削減のためにカメラで部屋を切り替えるごとにオブジェクトを有効化・無効化しているところでしょうか。
